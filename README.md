## 🎯 **Quick Start**  

1. **Clone the repo**  
   ```bash
   git clone https://github.com/saim-glitch/Freelancing.git
   ```
2. **Navigate to a project**  
   ```bash
   cd Freelancing/ai-phishing-detector
   ```
3. **Install dependencies**  
   ```bash
   pip install -r requirements.txt
   ```
4. **Run it!**  
   ```bash
   streamlit run app.py
   ```
